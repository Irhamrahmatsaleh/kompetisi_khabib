# kompetisi_khabib

Python OHLCV indicator analyzer for ATR, momentum, volatility, and trend review.

1. Clone the repository:
```bash
git clone https://github.com/Irhamrahmatsaleh/kompetisi_khabib.git
cd kompetisi_khabib
```

2. Install the dependencies:
```bash
sudo apt update && sudo apt install -y python3 python3-pip make
make install
```

3. Execute the engine:
```bash
make serve
```
